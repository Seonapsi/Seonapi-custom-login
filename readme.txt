Il plugin "Seonapsi Custom Login Style" personalizza la schermata di login di WordPress con il tuo stile.

== Description ==

Questo plugin, Seonapsi Custom Login Style, migliora l'esperienza di accesso al tuo sito WordPress personalizzando la pagina di login secondo
il tuo stile e il tuo marchio. 
Con un'interfaccia semplice e intuitiva, è possibile modificare facilmente l'aspetto della pagina di login senza dover toccare il codice.

== Caratteristiche principali ==

* Personalizzazione del logo: sostituisci il logo predefinito di WordPress con il tuo o con una immagine a tua scelta.
* Modifica dello sfondo: puoi cambiare colore alla pagina di login.
* Stili di testo: personalizza i colori del testo di campi e etichette.
* Stile dei pulsanti: modifica il colore, il bordo e gli effetti al passaggio del mouse sui pulsanti di accesso e di recupero password.
* Compatibilità con i principali browser e dispositivi: il plugin è stato testato e funziona perfettamente su tutti i principali browser e dispositivi moderni.

== Installazione ==

1. Scarica e installa il plugin "Seonapsi Custom Login Style" direttamente dalla directory dei plugin di WordPress o carica i file manualmente.
2. Attiva il plugin nella sezione "Plugins" nel tuo pannello di amministrazione WordPress.
3. Vai alla pagina delle impostazioni del plugin per iniziare a personalizzare la tua pagina di login, la trovi in Aspetto/Seonapsi Custom Login Style.
